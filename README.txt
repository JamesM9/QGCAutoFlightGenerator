AutoFlightGenerator v1.1.0
===========================

Professional UAV Mission Planning Application

NEW FEATURES IN v1.1.0:
- Master Settings System: Global unit preferences (Metric/Imperial)
- Enhanced Dashboard: Unified navigation with mission cards
- Mission Library: Save, load, and manage mission plans
- Status System: Real-time notifications and progress tracking
- Enhanced Map Integration: Advanced waypoint and route visualization
- Improved UI/UX: QGroundControl-inspired dark theme
- Altitude Profile Visualization: Generate detailed flight profiles
- Terrain Following: Automatic elevation data integration
- Aircraft Parameters System: Import ArduPilot/PX4 parameters for optimized flight planning

MISSION TYPES:
- Delivery Route: Single delivery missions with precise waypoints
- Multi-Delivery: Complex multi-point delivery routes
- Security Route: Security patrol missions with geofencing
- Linear Flight: Linear inspection and survey routes
- Tower Inspection: Tower and infrastructure inspection missions
- A-to-B Mission: Simple point-to-point mission planning

KEY FEATURES:
- Interactive Map: Click to set coordinates and view elevation
- Settings Management: Global preferences for units and defaults
- Mission Library: Save and organize mission plans
- Real-time Status: Progress tracking and notifications
- Professional UI: Dark theme with high contrast
- Export to QGroundControl: Generate .plan files
- Altitude Visualization: Detailed flight profiles
- Aircraft Parameters: Import and apply ArduPilot/PX4 parameters automatically
- Smart Optimization: Flight plans optimized for your specific aircraft
- Global Parameter Control: Enable/disable parameters from dashboard
- Video Tutorial System: Integrated instructional videos for all tools
- Comprehensive Learning: Step-by-step video guides for every feature
